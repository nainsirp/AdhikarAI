# AdhikarAI – Legal Helper for Women in India

AdhikarAI is a simple, easy-to-use tool built to help women navigate the Indian legal system. The law can feel scary and full of hard words, so this project is made to explain things in plain, everyday language (including regional languages) and give women the confidence they need.

Live website link: [https://nainsirp.github.io/AdhikarAI/](https://nainsirp.github.io/AdhikarAI/)

---

## What it does

Here are the main features we built:

* **Ask AI (Chatbot)**: Ask any legal question in simple English, Hindi, or other languages. You'll get clear answers citing the actual laws, without the confusing legal jargon.
* **Evidence Vault**: A safe place to upload and organize documents, photos, or audio files that you might need for your case. Files are saved securely in the browser so you don't lose them.
* **Procedure Mapper**: A step-by-step visual guide showing you exactly how things work in practice—like how to file an FIR at a police station or how court procedures go.
* **Legal Notice Drafter**: A tool that helps you create standard legal notices by filling out a simple form. You can select your state and city, fill in the details, and download a ready-to-use PDF.

## How to run it locally

Since this is a frontend-only static website, running it is very easy:

1. Clone the repository:
   ```bash
   git clone https://github.com/nainsirp/AdhikarAI.git
   ```
2. Open `index.html` in any web browser. (Or use a local server like Live Server in VS Code to run it).

---

## Technologies used

* **HTML5 & CSS3**: For the layouts, clean typography, and responsive designs.
* **Vanilla JavaScript**: For the interactive forms, state/city dropdowns, and browser storage handling.
* **Firebase**: Used to connect secure sign-in features and cloud storage.
* **html2pdf.js**: To let users download their drafted legal notices as clean PDF files.

---

## Why we built this

Finding legal help in India can be hard, expensive, and overwhelming. We wanted to build something that acts as a companion—a private, secure, and simple starting point for anyone who needs to understand their rights and take the first step.
